# 受験生のための過去問検索アプリ - MVP 完成

## 📦 納品物の内容

Django 5.x ベースの過去問検索・比較プラットフォームの完全な実装を納品します。

### ✅ 実装済み機能

#### 1. ユーザー認証機能
- ✅ ユーザー登録（サインアップ）
- ✅ ログイン/ログアウト
- ✅ マイページ（検索履歴・お気に入り管理）

#### 2. 過去問データ管理
- ✅ University モデル（大学・高校情報）

- ✅ AnswerSource モデル（予備校別解答）

- ✅ Bootstrap 5 によるレスポンシブデザイン

### 📊 データモデルの詳細

#### University（大学）
- 大学名、かな、学校種別、公式URL

#### Exam（過去問）
- 大学、年度、科目、試験種別
- 問題PDF URL、説明
- ソース種別、検証状態

#### AnswerSource（解答ソース）
- 過去問への紐付け
- 予備校名、解答URL
- 詳細解説の有無、動画解説の有無
- 信頼度スコア（1-10）

### 🚀 セットアップ手順（5分）

```bash
# 1. 仮想環境の作成
python -m venv venv
source venv/bin/activate  # Mac/Linux
# または venv\Scripts\activate  # Windows

# 2. パッケージのインストール
pip install Django==5.0

# 3. データベースの初期化
python manage.py migrate

# 4. スーパーユーザーの作成
python manage.py createsuperuser

# 5. サンプルデータの投入（オプション）
python manage.py loaddata exams/fixtures/sample_data.json

# 6. サーバー起動
python manage.py runserver
```

### 🎯 使用方法

1. **http://127.0.0.1:8000/** でトップページにアクセス
2. 新規登録またはログイン
3. 検索バーで大学名や科目を検索
4. 詳細ページで予備校別の解答を比較
5. お気に入りに保存してマイページで管理

### 📊 管理画面

**http://127.0.0.1:8000/admin/** で以下を管理:
- 大学データの追加・編集
- 過去問データの追加・編集
- 解答ソースの追加・編集
- ユーザー管理
- 検索履歴の確認

### 🔧 カスタマイズポイント

#### 1. デザインのカスタマイズ
`templates/base.html` の `<style>` タグ内でカラースキームを変更可能

#### 2. 科目の追加
`exams/models.py` の `SUBJECT_CHOICES` に科目を追加

#### 3. ページネーション件数
`exams/views.py` の `paginate_by = 12` を変更

#### 4. データベースの切り替え
`exam_search/settings.py` の `DATABASES` 設定を変更
